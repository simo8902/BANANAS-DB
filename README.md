# Bananas Dataset

This repository contains aggregated, non-actionable metadata about media categories such as movies, software, and games, collected for research and indexing purposes only

Sample data includes, but is not restricted to: categorization, filenames, and reference identifiers \
No links, access methods, hashes, trackers, or reconstruction data are included here, and this repository does not enable access to any external content

The dataset is intended solely for analytical use (statistics, categorization, naming patterns) \
For legitimate research inquiries, contact the maintainer privately
